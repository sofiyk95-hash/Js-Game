# 👾 Pixel-Art Arcade Browser Game

> A fully interactive, lightweight arcade browser game driven by vanilla JavaScript, custom CSS animations, and structured DOM manipulation. 

### 🎮 [Play the Live Demo](https://sofiyk95-hash.github.io/Js-Game/)

---

## 🚀 Features 
* **Smooth Game Mechanics:** Responsive controls and fluid character movement.
* **Dynamic Gameplay:** Automatic alien speed-up mechanics to increase difficulty over time.
* **Score Tracking:** Built-in scoring system to track your progress in real-time.
* **Retro Aesthetics:** Beautiful pixel-art visual style and custom fonts.

## 🛠️ Tech Stack 
* **Language:** JavaScript (ES6+)
* **Architecture:** Object-Oriented Programming (OOP)
* **Styling:** HTML5 / Custom CSS3 (Animations & Layout)
* **Code Quality:** Prettier (`.prettierrc`) for consistent formatting
